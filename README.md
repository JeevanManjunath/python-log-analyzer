# Python Log Analyzer

A Python-based security monitoring system that analyzes server logs to detect brute-force login attempts and visualize attack patterns.

## Features

- Log generation for testing
- Brute-force attack detection
- Real-time attack visualization
- Automated security reports
- Docker container support

## Technologies

Python
Matplotlib
Docker
Networking Concepts

## Run Project

python generate_logs.py
python log_analyzer.py
python attack_visualizer.py
